> 생성일: 2026-02-09

---

## 10회차 커리큘럼 한눈에 보기

| 회차 | 제목 | 핵심 키워드 | 주요 실습 |
|------|------|-------------|-----------|
| 1회차 | AI 시대 생존을 위한 첫걸음 - 환경이 절반이다 | PARA, Obsidian, Claude Code, Git | PARA 폴더 생성, Git 커밋 |
| 2회차 | 프롬프트 골든 공식 - 역할+맥락+지시+형식 | RCIF, Hallucination, 프롬프트 엔지니어링 | Good/Bad 프롬프트 비교, 업무 프롬프트 작성 |
| 3회차 | 24시간 협업 파트너 - Tool Use, Planning, Memory | AI Agent, ReAct, Agent Loop | Agent 설계 워크숍 |
| 4회차 | 24시간 일하는 나의 분신 만들기 | 백그라운드 자동화, 스케줄링, Skills | 일일 리포트 Agent 구축 |
| 5회차 | Obsidian - 제2의 뇌 구축 | PARA 심화, 양방향 링크, 플러그인 | 개인 지식 베이스 구축 |
| 6회차 | 구조화된 사고 → 자동 문서화 파이프라인 | RAG, MCP, 템플릿 자동화 | 자동 문서화 파이프라인 구축 |
| 7회차 | 1인 연구자의 무기들 | Personal Research Stack, Flywheel, nanobanana | 논문 리뷰 자동화 |
| 8회차 | 발표자료부터 데이터 분석까지 | 프레젠테이션, Excel 분석, 대시보드 | 연구 데이터 시각화 |
| 9회차 | 나만의 프로젝트 관리 시스템 | 멀티 에이전트, Vibe Kanban, Git Worktree | 멀티 에이전트 PR 플로우 |
| 10회차 | 외주가 아닌 직접 수행 - AI로 역할 유동성 확보 | T자형 역량, RAG 실전, 자동화 파이프라인 | 종합 실습, 수료 |

## 학습 흐름 (Phase별 연결)

```
[기초]  1회차 환경설정 → 2회차 프롬프트
   ↓
[개념]  3회차 Agent 이론 → 4회차 Agent 실전
   ↓
[도구]  5회차 Obsidian → 6회차 문서 자동화
   ↓
[응용]  7회차 연구 도구 → 8회차 데이터/발표
   ↓
[통합]  9회차 프로젝트 관리 → 10회차 종합 실습
```
