# Claude Code 사용법

## 기본 명령어

| 명령어 | 설명 |
|--------|------|
| `claude` | Claude Code 시작 |
| `claude --version` | 버전 확인 |
| `claude doctor` | 설치 상태 진단 |

## 대화 중 명령어

| 명령어 | 설명 |
|--------|------|
| `/help` | 도움말 보기 |
| `/init` | CLAUDE.md 파일 생성 (프로젝트 컨텍스트) |
| `/config` | 설정 확인/변경 |
| `/clear` | 대화 초기화 |
| `Shift+Tab` | Plan 모드 전환 |
