# Measurement Formats

## Standard Units and Conventions

### Pressure
- **Primary**: MPa (e.g., 0.16 MPa, 1.2 MPa)
- **Secondary**: bar (e.g., 28 bar, 4.3 bar)
- **In-cylinder**: MPa
- **Boost pressure**: bar or MPa
- Format: number + space + unit (e.g., "0.16 MPa")

### Temperature
- **Celsius**: °C (e.g., 8 °C, 500 °C)
- **Kelvin**: K (e.g., 2009 K)
- Combustion temperatures typically in K
- Intake/coolant temperatures in °C

### Engine Speed
- **Format**: number + space + rpm (e.g., "1600 rpm", "3000 rpm")
- Alternative: r/min

### Power
- **Primary**: kW (e.g., "73 kW", "5 kW")
- **Heavy-duty**: hp or PS (e.g., "780 hp", "600 hp")

### Emissions
- **Specific**: g/kWh (e.g., "0.4 g/kWh")
- **Concentration**: ppm (e.g., "200 ppm", "800 ppm")
- **Regulatory**: mg/kWh (e.g., "200 mg/kWh")

### Crank Angle
- **Primary**: CAD (crank angle degree)
- **Alternative**: °CA
- Example: "40 CAD", "12 °CA"

### Ratios
- **Compression ratio**: N:1 format (e.g., "15:1", "17.1:1")
- **Excess air ratio**: decimal (e.g., "2.2", "2.5")
- **Equivalence ratio**: decimal (e.g., "0.82")

### Efficiency / Percentage
- **Format**: number + % (e.g., "42.8%", "18.4%")
- Use one decimal place typically

### Flow Rate
- **Gas flow**: m3/h
- **Fuel flow**: kg/h or g/s

## Formatting Rules

1. Always place a space between number and unit: "0.16 MPa" not "0.16MPa"
2. Use consistent decimal places within a comparison
3. Define abbreviations at first use: "indicated thermal efficiency (ITE)"
4. Use parenthetical units in tables: "Pressure [MPa]" or "IMEP (MPa)"
5. Compression ratios use colon notation: "15:1" not "15 to 1"
