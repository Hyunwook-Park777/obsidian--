# Methodology / Experimental Setup Template

## Structure: Engine Setup → Instrumentation → Test Conditions → Data Processing

### Typical Subsections

#### 2.1 Engine Specifications / Test Engine
- Past tense, passive voice dominant
- Table with engine specifications (displacement, bore, stroke, compression ratio, etc.)
- "A [single/multi]-cylinder [engine type] was employed in this study."
- "Table [N] lists the engine specifications."
- "The test engine is a [description] based on a [platform]."

#### 2.2 Fuel System / Fuel Properties
- Describe fuel injection system
- Table with fuel properties (LHV, density, flammability limits, etc.)
- "The fuel was supplied to the engine through [injection type]."
- "[Fuel] was injected into the [location] using [injector type]."

#### 2.3 Experimental Apparatus / Instrumentation
- List measurement equipment
- Pressure sensors, temperature sensors, gas analyzers, etc.
- "In-cylinder pressure was measured using a [sensor type] pressure sensor."
- "The exhaust emissions were measured using a [analyzer type]."

#### 2.4 Test Conditions / Experimental Procedure
- Table or list of test parameters
- "The experiments were conducted at [speed] rpm under [load conditions]."
- "The [parameter] was varied from [X] to [Y] in increments of [Z]."
- "The engine was warmed up to [temperature] before data acquisition."

#### 2.5 Data Processing / Analysis Methods
- Describe calculation methods
- "The indicated thermal efficiency was calculated using Eq. ([N])."
- "An energy balance analysis was conducted to investigate..."
- "The coefficient of variation (COV) of IMEP was used as an indicator of combustion stability."

### Key Patterns
- Passive voice dominant: "was equipped", "was measured", "were recorded"
- Past tense for describing what was done
- Present tense for describing permanent setups: "The engine is equipped with..."
- Use figures for schematics: "Fig. [N] shows the schematic of the experimental setup."
- All instruments referenced with manufacturer/model when relevant
