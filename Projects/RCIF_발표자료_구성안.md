# RCIF 업무 프롬프트 #2 - 연구 성과 발표 슬라이드 구성안

## 실습 정보
- **업무 유형**: 연구 성과 발표 자료
- **연구 주제**: 무탄소 연료(암모니아) 엔진시스템의 디지털 트윈 구축
- **발표 대상**: 대한기계학회 추계학술대회
- **실습 일자**: 2026-02-07

---

## 1. 나쁜 프롬프트 (RCIF 미적용)

### 프롬프트

> "연구 발표자료 만들어줘"

### 결과

1. 제목 슬라이드
2. 연구 배경
3. 연구 목적
4. 연구 방법
5. 결과
6. 결론
7. 감사합니다

### 문제점
- **슬라이드 수 부족**: 7장으로는 15분 발표를 채울 수 없음
- **구체성 없음**: 어떤 연구인지, 어떤 학회인지, 청중이 누구인지 모름
- **시각 자료 제안 없음**: 각 슬라이드에 무엇을 넣어야 하는지 알 수 없음
- **핵심 메시지 없음**: 슬라이드 제목만 나열되어 발표 흐름을 잡을 수 없음
- 어떤 분야의 어떤 발표에나 적용 가능한 **일반적인 목차 수준**

---

## 2. RCIF 프롬프트 (골든 공식 적용)

### 사용한 RCIF 프롬프트

```
[R: 역할]
당신은 기계공학 분야 국제학회 발표 경험이 풍부한
연구 프레젠테이션 전문가입니다.

[C: 맥락]
나는 한국기계연구원 소속 연구원이며,
"무탄소 연료(암모니아) 엔진시스템의 디지털 트윈 구축" 과제의
1차년도 연구 성과를 학회에서 발표하려 합니다.
- 학회: 대한기계학회 추계학술대회
- 발표 형식: 구두 발표 15분 + 질의 5분
- 청중: 기계공학 분야 연구자 및 대학원생
- 핵심 성과: 암모니아 엔진 연소 모델과 실험 데이터를
  결합한 하이브리드 디지털 트윈 프로토타입 개발

[I: 지시]
발표 슬라이드 구성안(아웃라인)을 작성해주세요.
각 슬라이드마다 다음을 포함해야 합니다:
1. 슬라이드 제목
2. 핵심 메시지 (1문장)
3. 포함할 시각 자료 제안 (그래프, 다이어그램 등)
전체 슬라이드 수는 12~15장으로 구성해주세요.

[F: 형식]
마크다운 형식으로 슬라이드 번호별로 정리하고,
각 슬라이드는 "제목 / 핵심 메시지 / 시각 자료" 3항목으로
구분해주세요.
```

---

### 생성 결과

### 슬라이드 1: 표지
- **제목**: 암모니아 엔진시스템의 하이브리드 디지털 트윈 구축
- **핵심 메시지**: 무탄소 연료 엔진 개발을 가속화하는 디지털 트윈 플랫폼
- **시각 자료**: 과제명, 발표자 정보, 소속 로고, 학회명

### 슬라이드 2: 연구 배경 - 왜 무탄소 연료인가
- **제목**: IMO 2050 탄소중립과 무탄소 연료 전환
- **핵심 메시지**: 국제 해운 탄소중립 규제로 암모니아·수소 엔진 전환이 불가피하다
- **시각 자료**: IMO 규제 타임라인 그래프, 연료별 CO2 배출량 비교 막대 차트

### 슬라이드 3: 연구 배경 - 기존 개발 방식의 한계
- **제목**: 실험 기반 엔진 개발의 비효율
- **핵심 메시지**: 암모니아의 독성과 연소 불안정성으로 실험 중심 개발은 비용·시간이 과다하다
- **시각 자료**: 기존 개발 프로세스 흐름도, 비용·시간 소요 비교표

### 슬라이드 4: 연구 목표
- **제목**: 하이브리드 디지털 트윈 기반 엔진 개발 플랫폼
- **핵심 메시지**: 물리 모델 + AI 모델을 결합하여 엔진시스템 전체를 가상 환경에서 설계·검증한다
- **시각 자료**: 연구 목표 개념도 (물리 모델 ↔ AI 모델 ↔ 실험 데이터 삼각 구조)

### 슬라이드 5: 연구 방법론 개요
- **제목**: 하이브리드 디지털 트윈 아키텍처
- **핵심 메시지**: CFD 기반 물리 모델, 센서 데이터 기반 AI 보정, 실시간 동기화의 3계층 구조
- **시각 자료**: 시스템 아키텍처 다이어그램 (3계층 블록도)

### 슬라이드 6: 물리 기반 연소 모델
- **제목**: 암모니아 연소 반응 메커니즘 모델링
- **핵심 메시지**: 암모니아-공기 혼합기의 착화 지연, 화염 전파, NOx 생성을 정밀 모사한다
- **시각 자료**: 연소 반응 메커니즘 경로도, 온도-압력 조건별 시뮬레이션 컨투어

### 슬라이드 7: AI 보정 모델
- **제목**: 실험 데이터 기반 신경망 보정 모델
- **핵심 메시지**: 물리 모델의 예측 오차를 AI가 실시간으로 보정하여 정확도를 높인다
- **시각 자료**: 신경망 구조도, 보정 전/후 예측 정확도 비교 그래프

### 슬라이드 8: 실험 검증 셋업
- **제목**: 단일 실린더 엔진 시험 장치 구성
- **핵심 메시지**: 압력·온도·배기 센서를 통해 연소 특성 데이터를 실시간 수집한다
- **시각 자료**: 시험 장치 사진/개요도, 센서 배치도

### 슬라이드 9: 핵심 결과 1 - 연소 특성 예측
- **제목**: 암모니아 엔진 연소압력 예측 결과
- **핵심 메시지**: 하이브리드 모델이 순수 물리 모델 대비 예측 오차를 대폭 줄였다
- **시각 자료**: 실험 vs 물리 모델 vs 하이브리드 모델 연소압력 곡선 비교 (선 그래프)

### 슬라이드 10: 핵심 결과 2 - NOx 배출 예측
- **제목**: NOx 배출량 예측 정확도
- **핵심 메시지**: 다양한 운전 조건에서 NOx 배출을 정밀 예측하여 후처리 설계에 활용 가능하다
- **시각 자료**: 운전 조건별 NOx 예측 vs 실측 산점도, 오차율 표

### 슬라이드 11: 핵심 결과 3 - 실시간 시뮬레이션
- **제목**: 실시간 디지털 트윈 운용 시연
- **핵심 메시지**: 과도 운전 조건에서도 실시간 연소 상태 모니터링 및 이상 예측이 가능하다
- **시각 자료**: 실시간 대시보드 스크린샷, 응답 시간 비교 막대 차트

### 슬라이드 12: 기존 연구 대비 차별성
- **제목**: 기존 접근법과의 비교
- **핵심 메시지**: 단일 연소실 CFD 수준을 넘어, 시스템 전체를 실시간으로 다루는 점이 차별적이다
- **시각 자료**: 기존 연구 vs 본 연구 비교표 (범위, 속도, 정확도)

### 슬라이드 13: 향후 계획
- **제목**: 2~3차년도 연구 로드맵
- **핵심 메시지**: 다기통 엔진 확장, 현장 실증, 자율 운전 최적화로 발전시킨다
- **시각 자료**: 연차별 마일스톤 타임라인 (간트 차트 형태)

### 슬라이드 14: 결론
- **제목**: 결론 및 기대효과
- **핵심 메시지**: 하이브리드 디지털 트윈은 무탄소 연료 엔진 개발의 비용과 시간을 획기적으로 줄일 수 있다
- **시각 자료**: 핵심 성과 3가지 요약 인포그래픽

### 슬라이드 15: Q&A
- **제목**: 감사합니다 / 질의응답
- **핵심 메시지**: -
- **시각 자료**: 연락처, 논문 인용 정보, QR코드(관련 자료 링크)

## 3. 나쁜 프롬프트 vs RCIF 비교

| 항목 | 나쁜 프롬프트 | RCIF 프롬프트 |
|------|-------------|--------------|
| **슬라이드 수** | 7장 (부족) | 15장 (적절) |
| **구체성** | 제목만 나열 | 제목 + 핵심 메시지 + 시각 자료 제안 |
| **맥락 반영** | 어떤 발표에나 해당 | 암모니아 엔진, 대한기계학회, 15분 발표에 맞춤 |
| **실용성** | 그대로 쓸 수 없음 | 슬라이드 제작 가이드로 바로 활용 가능 |

## 4. 프롬프트 #1 vs #2 비교

| 항목 | 프롬프트 #1 (연구 제안서) | 프롬프트 #2 (발표 자료) |
|------|-------------------------|------------------------|
| **R (역할)** | 제안서 심사 전문가 | 학회 발표 전문가 |
| **C (맥락)** | 산업부 사업, 30억, 3년 | 대한기계학회, 15분 발표, 1차년도 성과 |
| **I (지시)** | 연구 필요성 섹션 작성 | 슬라이드 구성안 작성 |
| **F (형식)** | 마크다운, 1500자, 굵은 수치 | 슬라이드별 제목/메시지/시각자료 |
| **결과물** | 제안서 본문 텍스트 | 15장 슬라이드 아웃라인 |

> **같은 RCIF 공식이지만 각 요소를 바꾸는 것만으로 완전히 다른 결과물이 생성된다.**
> 이것이 RCIF 공식의 유연성이자 핵심 가치이다.
