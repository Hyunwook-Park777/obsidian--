## 2. 핵심결과 (1) - 핵심 요소기술 개발

### 2-1. 냉형 점화플러그 적용

| 항목 | 베이스 점화플러그 | 냉형 점화플러그 |
|:---|:---|:---|
| 이른 조기점화 | 고부하에서 빈번 발생 → 순간 출력 하락 | **완전 억제** |
| 늦은 조기점화 | 다수 발생 | 발생빈도 대폭 감소 |
| 백금 제거 효과 | - | 늦은 조기점화 추가 소폭 감소 |
| 적용 결과 | 100% 부하 운전 제한 | **고부하 안정 운전 확보** |

> 연소실 내 주요 열점인 점화플러그 온도를 낮추어, 공기과잉률 2.0~2.2 범위에서 이른 조기점화를 완전히 억제

### 2-2. 포트분사 인젝터 사양 선정

| 평가 항목 | 작은 구경 인젝터 | 큰 구경 인젝터 (선정) |
|:---|:---|:---|
| 아이들 연소안정성 | 유리 | **COV of IMEP < 3% 달성** |
| 정격출력 분사 인가기간 | 과도하게 길어 이상연소 유발 | **< 10ms 만족** |
| 300kW 정격출력 대응 | 불가 (연료 공급 부족) | **충분한 연료 공급** |

> 정격출력(300kW) 달성에 필요한 최대 분사 인가기간과 아이들 연소안정성을 동시에 만족하는 큰 구경 인젝터 선정

### 2-3. 피스톤 형상 최적화

| 사양 | 압축비 | Squish 영역 | 특성 |
|:---:|:---:|:---:|:---|
| Base (GX12) | 10:1 | 기본 | 천연가스 엔진 기준 설계 |
| **Type A** | **11:1** | 기본 | 열효율 ~1%p 향상, 조기점화 없음 |
| Type B | 11:1 | 좁음 | 연소효율 소폭 증가, 고부하 노킹 취약 |
| **Type C (최종)** | **11:1** | **넓음** | Squish 유동 강화 → **조기점화+노킹 동시 억제** |

> - 열효율은 피스톤 형상보다 **압축비가 지배적** → 11:1 최적 선정
> - 고부하 이상연소 억제는 **Squish 유동 강화가 핵심** → Type C 제안
> - 1,800rpm 100%부하: **40.4%**, 1,200rpm 100%부하: **41.4%** 열효율 달성

## 3. 핵심결과 (2) - 통합제어 전략 및 고성능화

### 3-1. 운전영역별 연소-후처리 통합제어 전략

#### 제어변수 영향 분석

| 운전영역 | 열효율 지배 인자 | NOx 제어 | 제한 요소 |
|:---|:---|:---|:---|
| 중속 저부하 | **공기과잉률** 지배적 | 높은 λ로 저감 | - |
| 중속 중/고부하 | 연소상 + 공기과잉률 동시 | 높은 λ + 지각 연소상 | NOx 배출 |
| 고속 저/중부하 | 연소상 + 공기과잉률 동시 | 낮은 λ 적용 가능 | - |
| 고속 고부하 | **연소상** 지배적 | 낮은 λ + 지각 연소상 | 노킹, 조기점화 |

#### 최종 통합 전략

| 엔진 부하 | 중속 전략 | 고속 전략 |
|:---:|:---|:---|
| **저부하** | 높은 공기과잉률 + 최적 연소상 → **최고 열효율** | 낮은 공기과잉률 + 최적 연소상 |
| **고부하** | 높은 공기과잉률 + 지각된 연소상 → **NOx 억제** | 낮은 공기과잉률 + 지각된 연소상 → **출력 확보** |

> - **중속**: 높은 Engine-out NOx → 높은 공기과잉률로 대응, 저부하는 최적 연소상, 고부하는 지각 연소상
> - **고속**: 유동 강화로 낮은 NOx → 낮은 공기과잉률 활용, 고부하에서 노킹/조기점화 고려 점화시기 지각
> - **후처리**: 전 영역 TC 후단 400℃ 이상 → Urea SCR 변환효율 99%, **Euro 7 충족**
> - **후처리 간소화**: 디젤 DOC+DPF+SCR(3단) → 수소 **DOC+SCR(2단)**, DPF 삭제 + 사이즈 축소

### 3-2. 2-Stage 터보차저 적용 및 고성능화

#### Single VGT 대비 성과

| 엔진 속도 | 최고토크 향상 | 핵심 제한 요소 | 최적화 전략 |
|:---:|:---:|:---|:---|
| **저속** | **+42.6%** | 조기점화, 낮은 배기에너지 | 점화시기 지각 + 적정 당량비 선정 |
| **중속** | **+6.1%** | 인터쿨러 전단 온도, NOx | 고정 과급압에서 당량비 최적화 |
| **고속** | **+3.4%** | 인터쿨러 전단 온도, 조기점화 | NOx 마진 활용한 당량비 상향 |

#### 저속 토크 향상 메커니즘

```
점화시기 지각 → 배기에너지 확보 → 과급압력 형성 (가장 효과적)
    └─ 한계: 조기점화 발생 (점화시점 실린더 압력/온도 상승)

당량비 증가 → 수소 유입 증가 → 토크 상승
    └─ 낮은 당량비: VGT duty 증가 → 배기압 상승 → 잔여가스↑ → 조기점화
    └─ 높은 당량비: 연소온도 상승 → 조기점화 재발생
    └─ 적정 당량비에서 조기점화 최소화 + 열효율 최적화
```

#### 향후 고성능화 로드맵

| 과제 | 기술 방안 | 기대 효과 |
|:---|:---|:---|
| 저속토크 추가 향상 | **하이브리드화** | 디젤 대비 낮은 저속토크/반응성 보완 |
| 중/고속 토크 향상 | **2단 인터쿨러** (mid charge air cooling) | 인터쿨러 전단 온도 한계 극복 |
| 안전성 확보 | **Blowby 열점 관리** | 5% 이상 수소 농도 → 관련 장치/유로 관리 |
