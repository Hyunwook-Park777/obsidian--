## 2. 핵심결과 (1) - 엔진 성능 달성

### 2-1. 출력 및 열효율

| 항목 | 달성 결과 |
|:---|:---|
| 최고출력 | **300kW** (2,000 rpm) |
| 최고 열효율 | **41.4%** (1,200 rpm 100% 부하, 압축비 11:1) |
| 연소효율 | **전 영역 99% 이상** |
| 천연가스 대비 열효율 개선 | +2.8%p (연료 변경만으로) |

### 2-2. 2-Stage 터보차저 적용 성과

| 엔진 속도 | Single VGT 대비 최고토크 향상 | 제한 요소 |
|:---:|:---:|:---|
| 저속 | **+42.6%** | 낮은 배기에너지, 조기점화 |
| 중속 | **+6.1%** | 인터쿨러 전단 온도, NOx 배출 |
| 고속 | **+3.4%** | 인터쿨러 전단 온도, 조기점화 |

### 2-3. 이상연소 억제 전략 체계

```
역화 억제 ─── 공기과잉률 증가 (λ ≥ 2.0)
조기점화 억제 ─ 냉형 점화플러그 + 수소 분사시기/압력 최적화
노킹 억제 ─── 최적 압축비(11:1) + 점화시기 지각
종합 제어 ─── EGR 적용 (저부하) + 공기과잉률 조절 (전 영역)
```

## 3. 핵심결과 (2) - 배출 및 통합 전략

### 3-1. 배출가스 특성

| 배출물 | 수소 엔진 특성 | 대응 전략 |
|:---|:---|:---|
| **NOx** | 유일한 유해 배출물 | 공기과잉률 증가 + Urea SCR (변환효율 99%) |
| **CO / CO₂** | 제로 수준 | 무탄소 연료 적용 효과 |
| **THC** | 오일 연소로 일부 발생 | 오일 유량 제어, 피스톤/라이너 간극 최적화 |
| **PM** | 제로 수준 | DPF 불필요 → 후처리 간소화 |

### 3-2. 운전영역별 통합 제어 전략

| 엔진 부하 | 중속 전략 | 고속 전략 |
|:---:|:---|:---|
| **저부하** | 높은 공기과잉률 + 최적 연소상 | 낮은 공기과잉률 + 최적 연소상 |
| **고부하** | 높은 공기과잉률 + 지각된 연소상 | 낮은 공기과잉률 + 지각된 연소상 |

> **전략 핵심**: 중속은 높은 Engine-out NOx 대응 → 높은 공기과잉률 적용
> 고속은 낮은 Engine-out NOx 활용 → 낮은 공기과잉률로 출력 확보

### 3-3. 후처리시스템 간소화

```
기존 디젤엔진:  DOC → DPF → SCR  (3단 구성)
수소엔진:       DOC → SCR          (2단 구성, DPF 삭제)
                + 사이즈 축소 가능 (낮은 NOx 배출 활용)
```

### 3-4. 향후 개발 방향

- **하이브리드화**: 디젤 대비 낮은 저속토크 및 반응성 보완
- **2단 인터쿨러**: Mid charge air cooling 적용 → 중/고속 추가 토크 향상
- **Blowby 관리**: 5% 이상 수소 농도 → 열점 관리 필수
