# Agent화 설계 문서: 엔진 시험 압력 데이터 자동 추출 Agent

> **작성자**: 박현욱
> **작성일**: 2026-02-19
> **과제**: Phase 2 - Claude Code 자동화 태스크

---

## 1. 프로젝트 개요

### 프로젝트명
엔진 다이노 시험 데이터 압력 지표 자동 추출 Agent

### 목표
엔진 성능 시험 결과 Excel 파일(86~88개 컬럼)에서 **압력 관련 지표 9개**를 자동으로 추출하여 하나의 통합 Excel 파일로 생성

### 예상 효과
- 기존 소요 시간: **15~20분** (파일별 수작업 복사/붙여넣기)
- 자동화 후 예상 시간: **1~2분** (명령 입력 + 결과 확인)
- 절감 시간: **약 15분/회**
- 추가 효과: 수작업 실수(컬럼 누락, 행 불일치) 방지

---

## 2. 업무 분석

### 2.1 업무명
엔진 다이노 시험 데이터에서 압력 관련 지표 추출

### 2.2 현재 수행 방식
- **소요 시간**: 시험 1회당 15~20분
- **반복 주기**: 시험 수행 시마다 (주 3회 이상)
- **수작업 내용**:
  1. 시험 장비에서 출력된 .xls 파일 열기 (86~88개 컬럼)
  2. 필요한 9개 압력 컬럼을 눈으로 찾기
  3. 해당 컬럼 데이터를 새 Excel 파일에 복사/붙여넣기
  4. 여러 시험 파일이 있으면 반복 수행
  5. 출처(파일명) 정보 수동 기입

### 2.3 입력 정보
- 엔진 다이노 시험 결과 Excel 파일 (.xls)
  - 파일당 86~88개 컬럼, 5~13행의 시험 데이터
  - 1행째에 단위 정보 헤더(kPa 등) 포함
- 추출 대상 압력 컬럼 9개:
  - `IntakeAirPress_Avg` (흡기 압력)
  - `P_uComp_Avg` (컴프레서 상류 압력)
  - `P_mComp_Avg` (컴프레서 중간 압력)
  - `P_Boost_uIC_Avg` (인터쿨러 상류 부스트 압력)
  - `P_Boost_dIC_Avg` (인터쿨러 하류 부스트 압력)
  - `P_Boost_Intake_Avg` (흡기 부스트 압력)
  - `P_uTC_Avg` (터보차저 상류 압력)
  - `P_mTC_Avg` (터보차저 중간 압력)
  - `P_dTC_Avg` (터보차저 하류 압력)

### 2.4 출력 결과물
- 파일명: `Extracted_Pressure_Data.xlsx`
- 형식: Excel (.xlsx)
- 내용: Source_File, Time + 압력 9개 컬럼, 전체 파일 통합

### 2.5 자동화 가능 부분
- 폴더 내 .xls 파일 자동 탐색
- 지정 컬럼 자동 추출
- 단위 헤더행 자동 제거
- 다수 파일 통합 병합
- 출처(파일명) 자동 태깅
- 결과 Excel 파일 자동 생성

### 2.6 사람이 해야 할 부분
- 추출할 컬럼명 지정
- 최종 결과물 검토 및 확인
- 필요시 추가 분석 수행

## 3. Agent 설계

### 3.1 입력 (Input)
```
- Data 폴더 경로 (엔진 시험 .xls 파일이 있는 폴더)
- 추출 대상 압력 컬럼명 목록 (사용자 지정)
```

### 3.2 처리 과정 (Process)
```
1. Data 폴더 스캔 → .xls 파일 목록 확인
2. 각 파일의 컬럼 구조 확인 (대상 컬럼 존재 여부 검증)
3. 필요한 라이브러리 확인 (pandas, openpyxl, xlrd)
4. 각 파일에서 지정 컬럼 추출 + 단위 헤더행 제거
5. 출처(Source_File) 컬럼 추가
6. 전체 파일 데이터 통합 (concat)
7. 결과를 .xlsx 파일로 저장
8. 결과 검증 (행 수, 통계 요약 출력)
```

### 3.3 출력 (Output)
```
- 파일명: Extracted_Pressure_Data.xlsx
- 형식: Excel (.xlsx)
- 컬럼: Source_File, Time, IntakeAirPress_Avg, P_uComp_Avg,
        P_mComp_Avg, P_Boost_uIC_Avg, P_Boost_dIC_Avg,
        P_Boost_Intake_Avg, P_uTC_Avg, P_mTC_Avg, P_dTC_Avg
```

## 4. 프롬프트 설계

### Agent 지시 프롬프트
```
Data 폴더 내의 3개의 엑셀 파일을 읽고,
IntakeAirPress_Avg, P_uComp_Avg, P_mComp_Avg, P_Boost_uIC_Avg,
P_Boost_dIC_Avg, P_Boost_Intake_Avg, P_uTC_Avg, P_mTC_Avg, P_dTC_Avg
의 순서로 압력 지표만을 추출하여 통합 Excel을 만들어줘.
```

### 프롬프트 설계 의도
- **간결한 자연어 지시**: 복잡한 코드나 절차를 지시하지 않고, "무엇을 해달라"만 전달
- **Agent가 스스로 판단할 부분**: 폴더 탐색, 라이브러리 확인, 파일 구조 분석, 단위 헤더 제거, 통합 방식
- **핵심**: Agent Loop(계획 → 실행 → 검토 → 반복)를 통해 Agent가 자율적으로 문제를 해결
